CorelDRAW 2018 v20.0.0.633 官方正式版

  • 资源

前言

  矢量图形设计软件CorelDRAW Graphics Suite 2018 版本于2018年4月初推出，Corel公司官网CorelDRAW 2018 在本月中旬已经全球正式发售，国内售价8200元人民币，感兴趣的可以体验。2018 版本带来了全新对称绘图模式、图块阴影工具、位图透视、以及支持发布至WordPress等令人印象深刻的创意功能！
  CorelDRAW是国外加拿大Corel公司开发的一款功能强大的专业平面设计软件、矢量设计软件、图形设计软件。其非凡的设计能力广泛地应用于商标设计、标志制作、封面设计、CIS设计、产品包装造型设计、模型绘制、插图描画、时装/服饰设计、印刷制版、排版及分色输出等诸多领域。20多年的历史其被广泛应用的程度足以说明，其用户涵盖图形设计、平面设计、图文设计、广告设计、商业设计和美术设计等多个领域行业。




CorelDRAW 2018 零售版包含组件：
CorelDRAW 2018 – 矢量插图和页面布局
Corel® PHOTO-PAINT® 2018 – 位图插图和图像编辑
Corel Font Manager™ 2018 – 字体开发与管理工具
Corel® PowerTRACE™ 2018 – 位图转矢量跟踪（CorelDRAW应用程序中随附）
Corel® CONNECT™ 2018 – 内容查找器和管理器
Corel® CAPTURE™ 2018 – 屏幕捕获工具
Corel® Website Creator™ – 网站设计
CorelDRAW 2018 支持的应用程序：
BenVISTA PhotoZoom Pro 7 — 用于放大数码图像的插件
Microsoft Visual Basic for Applications 7.1– 任务自动执行
Microsoft Visual Studio Tools for Applications 2017 – 任务自动执行和高级宏项目
Barcode Wizard – 行业标准格式条形码生成器
Duplexing Wizard – 双面打印指南
GPL Ghostscript (ZIP) – 增强型 EPS 和 PS 文件导入
WhatTheFont – 在线字体标识符
